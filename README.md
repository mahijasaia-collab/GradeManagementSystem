# Student Grade Management System 🚀

A structured, Object-Oriented Java console application designed to handle student data, calculate key academic statistics, and maintain data across sessions using local file persistence.

## 🌟 Features
* **Full CRUD Operations:** Add, update, and delete student profiles seamlessly.
* **Smart Performance Metrics:** Uses Java Streams to calculate class averages, determine highest/lowest grades, and identify class toppers.
* **Grade Distribution:** Dynamically groups numerical scores into standard letter grades (A, B, C, D, F).
* **Data Persistence:** Automatically saves records to a local text database (`students.txt`) and reads them back upon startup.

## 🛠️ Technologies Used
* **Language:** Java (OpenJDK 21)
* **Core Concepts:** Object-Oriented Programming (OOP), Java Streams API, File Collections, Input/Output Exception Handling.
